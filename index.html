About the Banking System

The Online Banking System is a web-based application designed to provide users with secure and efficient banking services. This system allows users to manage their bank accounts digitally without visiting a physical bank.


The project is developed using java with SpringBoot Framework and follows Object-Oriented Programming principles to ensure scalability, security, and maintainability.

Key Features

Account Creation

Create a new bank account with initial balance.

Deposit Money

Securely deposit funds into your account.

Withdraw Money

Withdraw money with balance validation.

Check Balance

View real-time account balance.

Update Account Details

Change account holder name securely.

Dark / Light Mode

User-friendly theme toggle for better experience.
